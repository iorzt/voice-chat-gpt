# Voice-Chat-GPT
chat with ChatGPT(LLM) by VOICE on your laptop(MacOS)

## ENV
+ set up OpenAI key 
  ```bash 
  export  OPENAI_API_KEY=sk-xx 
  ```
+ install requirements
  ```
     // set up venv
     python -m venv venv
     // install dependencies
     pip install -r requirements.txt
  ```

## RUN

```
python main.py
```
